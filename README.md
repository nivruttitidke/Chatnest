ChatNest — Real-Time Chat & Video Calling Platform

ChatNest is a full-stack real-time communication application built using the MERN Stack, featuring secure authentication, instant messaging, and one-click video calling.
The application is deployed in production and demonstrates end-to-end full-stack development, real-time systems integration, and cloud deployment practices.
